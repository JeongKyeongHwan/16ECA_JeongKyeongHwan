1. add_mat() 함수에서 행렬의 덧셈 전 어떤 확인 과정이 있으면 좋을 것인가?

X = a x b 행렬이고 Y = c x d 행렬일때, a = c, b = d 임을 확인해야한다.
 
2. mul_mat_vec() 함수에서 행렬 벡터 곱셈 전 어떤 확인 과정이 있으면 좋을 것인가?

X = a x b 행렬이고 Y = c x d 행렬일때, XY일 경우에는 b = c, YX일 경우에는 d = a 임을 확인해야한다.
 
3. 행렬식 함수에서는 함수 자신을 호출하는데 이 것을 무엇이라고 부르는가?

재귀호출

4. Python 에서는 alloc_mat_bad_00() 또는 alloc_mat_bad_01() 과 같이 행렬을 생성하는 것이 바람직한가?  그 까닭은?


바람직하지 않다. 왜냐하면 이름이 비슷할 경우 함수를 구성하다가 stack overflow를 발생시킬 위험이 있다.
